Welcome to the buggy race editor

This buggy editor, allows buggies to be edited and created in order to be entered into a race against other buggies.

This editor buggy has the following features :

  • Multiple buggies can be created with the following customisable features:
    • flag colour
    • a secondary flag colour
    • a flag pattern
    • hamster booster quantity
  • Edit aspects each buggy created
  • Retrieve JSON data so buggies can be entered into races
